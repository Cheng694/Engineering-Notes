# WSL中安装miniconda
### 1 下载
```bash
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh -O miniconda.sh
```

### 2 下载后给予权限
```bash
chmod 777 Miniconda3-latest-Linux-x86_64.sh
```
### 3 运行安装软件
```bash
bash Miniconda3-latest-Linux-x86_64.sh
```
![alt text](image.png)

### 4 关闭并重新打开终端，或运行以下命令生效:
```bash
source ~/.bashrc
```
### 5 验证安装
```bash
conda --version
```
